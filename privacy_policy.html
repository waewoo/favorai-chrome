FavorAI — Politique de confidentialité

Dernière mise à jour : mai 2026 · Version de l'extension : 1.1.0

✅ En résumé : FavorAI ne collecte aucune donnée personnelle pour son propre compte. Il ne possède aucun serveur, aucune base de données, aucune télémétrie. Votre clé API et vos favoris ne quittent jamais votre appareil sauf pour être envoyés au service d'IA que vous avez vous-même configuré.

1. Qui sommes-nous ?

FavorAI est une extension de navigateur développée à titre personnel / indépendant. Elle est publiée sur le Microsoft Edge Add-ons Store et le Chrome Web Store. Le code source est disponible pour audit.

3. Transmission de données à votre service d'IA

⚠️ Point important : Lorsque vous lancez une réorganisation, FavorAI construit un fichier JSON contenant les titres et la structure hiérarchique de vos favoris, puis l'envoie au service d'IA que vous avez configuré dans l'onglet Configuration (OpenAI, Google Gemini, Mistral, Grok, Ollama ou un endpoint personnalisé).

FavorAI n'a aucun contrôle sur la politique de confidentialité de ces services tiers. Nous vous encourageons à consulter la politique de confidentialité du fournisseur que vous utilisez :

  • OpenAI — Politique de confidentialité
  • Google (Gemini) — Politique de confidentialité
  • Mistral AI — Politique de confidentialité
  • xAI (Grok) — Politique de confidentialité
  • Ollama (local) — Aucune transmission externe ; le modèle s'exécute entièrement sur votre machine

4. Ce que FavorAI ne fait jamais

  • Nous n'avons aucun serveur propre — FavorAI n'a pas de backend, pas de base de données, pas de compte utilisateur
  • Nous ne collectons aucune donnée à des fins d'analyse, de publicité ou de profilage
  • Nous n'utilisons aucune telémétrie, aucun tracker, aucun pixel de suivi
  • Nous ne partageons aucune donnée avec des partenaires tiers autres que le LLM que vous avez vous-même configuré
  • Nous ne stockons pas les URLs de vos favoris — elles restent sur votre appareil
  • Nous ne lisons pas vos onglets ouverts, votre historique de navigation, vos cookies ou vos mots de passe
  • Nous n'injectons aucun script dans les pages que vous visitez

Permission Usage

<all_urls> Deux usages : (1) vérifier si les URLs de vos favoris sont accessibles (liens morts) en envoyant des requêtes HEAD/GET aux sites concernés, et (2) communiquer avec l'API LLM que vous avez configurée, qui peut pointer vers n'importe quel domaine (y compris un serveur local)

6. Sécurité de votre clé API

Votre clé API est stockée dans chrome.storage.sync. Ce stockage est chiffré par le navigateur et synchronisé entre vos appareils connectés au même compte Microsoft/Google Edge/Chrome. FavorAI ne peut pas lire votre clé en dehors du contexte de l'extension.

Recommandation : Créez une clé API dédiée à FavorAI avec un quota limité chez votre fournisseur LLM.

7. Vos droits (RGPD)

Puisque FavorAI ne collecte aucune donnée personnelle pour son propre compte, les droits d'accès, de rectification et de suppression s'appliquent directement à votre navigateur :

  • Supprimer vos données : Utilisez le bouton « Vider » dans l'onglet Historique, ou désinstallez l'extension (toutes les données locales sont effacées)
  • Exporter votre configuration : Utilisez le bouton « Exporter » dans l'onglet Configuration
  • Exercer vos droits auprès du LLM : Contactez directement votre fournisseur d'IA (OpenAI, Google, Mistral, etc.) pour les données qu'ils ont pu traiter

8. Modifications de cette politique

Toute modification significative de cette politique sera signalée dans les notes de version de l'extension. La date de dernière mise à jour est indiquée en haut de cette page.
