FavorAI

Intelligent bookmark management

✨ What is FavorAI?

FavorAI is a Chrome extension that uses artificial intelligence to intelligently organize your bookmarks. It analyzes your bookmark structure, detects smart duplicates and dead links, then proposes or applies semantic reorganization.

🎯 Core Features

🔍 Smart duplicate detection

🔗 Dead link detection

🤖 AI-powered organization

📜 Organization history & rollback

🔌 Support for multiple AI providers

🔗 Supported Providers

🟠 OpenAI
🔵 Google Gemini
🟣 Mistral
⚡ Claude
🟢 DeepSeek
🤖 Grok
🏠 Ollama

🔒 Privacy & Security

  • ✓ Only titles and folder structure (no URLs) are sent to AI providers
  • ✓ Your API key is stored locally in your browser
  • ✓ No data is sent to FavorAI servers
  • ✓ Analysis happens in your browser
  • ✓ You maintain full control over your bookmarks

💡 Quick Tips

👉 Getting Started
1. Go to Configuration tab and select your LLM provider
2. Enter your API key (get one free from any provider)
3. Click "Minimal" to start your first reorganization
4. Review changes in the checklist before applying
⚙️ Configuration Tips
• Try different providers — they suggest different organizations
• Enable "Check dead links" for thorough analysis
• Use "Minimal" mode to review changes first
• Check History tab to rollback changes if needed

⚙️ How It Works
