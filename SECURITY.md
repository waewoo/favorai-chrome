# Security Policy

## Supported versions

Security fixes are applied to the current maintained version of FavorAI.

## Reporting a vulnerability

Please do not open a public issue for security-sensitive findings.

Send a private report to:

- Email: `ftzpr-chrome@yahoo.com`

Include as much detail as possible:

- The affected file or feature
- Steps to reproduce
- Any relevant screenshots, logs, or proof of concept
- Whether the issue could expose user data, API keys, or bookmark content

## What to expect

I will acknowledge the report, validate the issue, and coordinate a fix or mitigation as needed.

Please avoid sharing exploit details publicly until a fix is available.
