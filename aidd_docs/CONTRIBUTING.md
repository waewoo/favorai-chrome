# Contributing to the AI Context

How to change the context the AI relies on here.

## Changing project memory

- Add or edit a file under `aidd_docs/memory/`.
- Keep each memory file focused on one concern and current state only.
- Update the root `AGENTS.md` memory block after changing the memory set.

## House conventions

- Prefer intention-first names.
- Keep memory files small and non-derivable.
- Do not add speculative sections or future TODOs.
