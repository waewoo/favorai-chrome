# VCS

## Setup

- Main branch: `main`
- Platform: GitHub
- Ticketing: none

## Branches

- Format: `codex/<short-description>`
- Types in use: `feat`, `fix`, `chore`, `docs`, `refactor`, `test`

## Commits

- Convention: Conventional Commits
- Format: `type(scope): description`
- Rules: imperative mood, lowercase summary, keep it atomic
